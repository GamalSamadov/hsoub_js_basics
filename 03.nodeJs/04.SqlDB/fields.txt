مرحباً جمال،

في Sequelize في Node.js، يمكنك استخدام الدوال التالية في Sequelize لبناء أنواع مختلفة من العلاقات:

- One-to-One: استخدام `hasOne` و `belongsTo`.
- One-to-Many: استخدام `hasMany` و `belongsTo`.
- Many-to-Many: استخدام `belongsToMany`.

على سبيل المثال، إذا كنت ترغب في إنشاء علاقة بين جدول `User` و `Post`، يمكنك استخدام الكود التالي:

const User = sequelize.define('User', {
  // سمات المستخدم
});

const Post = sequelize.define('Post', {
  // سمات المنشور
});

User.hasMany(Post);
Post.belongsTo(User);

هذه الدوال تسمح لك بتحديد العلاقات بين نماذجك في Sequelize، بطريقة مشابهة لكيفية تحديدها في Django باستخدام دوال العلاقات.
